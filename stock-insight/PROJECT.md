# StockInsight Pro — A股全链路量化分析平台

## 基本信息
- **仓库**: https://github.com/nguyenchunghieu799-blip/stock-insight
- **技术栈**: Python 3.x / FastAPI / React / Tauri / SQLite / Chart.js
- **定位**: 借鉴参考项目，为 Claw / QTS 优化提供参考方案

## 核心能力（7层分析）
| 层级 | 内容 |
|:---:|------|
| L0 | K线数据 & 实时行情（7源容灾） |
| L1 | 技术指标（MACD/RSI/KDJ/布林/27种K线形态） |
| L2 | 基本面（ROE/PE/PB + 公司质地七问） |
| L3 | 量化多因子评分（动量/技术/基本面/量能/风险/舆情/资金流） |
| L4 | 实时行情 & 国家队/北向/龙虎榜 |
| L5 | 资金流向 & 短线评分 |
| L6 | ML集成预测（XGBoost+RF+LGB投票） |
| L7 | 宏观 & 回测（7大策略对比） |

## 值得借鉴的特性
1. **多源数据容灾** — 7个K线数据源，优先、回退、容错
2. **三级缓存** — 内存 → SQLite → API，宏观数据查询从23.8秒→0.006秒
3. **增强选股器** — 大盘→板块→个股三过滤 + 双轮筛选
4. **27种K线形态识别** — 含中文解读和置信度
5. **庄家博弈四阶段** — 建仓/洗盘/拉升/出货自动识别
6. **多空辩论** — 基于规则的NLP看涨/看跌论点生成
7. **HTML交互报告** — Chart.js雷达图+评级分布+因子对比
8. **系统自审计** — 6大健康检查项
9. **Tauri桌面端** — React + FastAPI + Rust三进程
